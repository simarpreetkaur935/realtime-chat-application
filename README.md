A full-stack real-time chat application built using the **MERN Stack**. The application allows users to register, log in securely, and communicate instantly through one-to-one messaging using **Socket.IO**.

- 🔐 Secure User Authentication (JWT)

- 💬 One-to-One Real-Time Messaging
- ⚡ Instant Message Updates with Socket.IO

- 🛡️ Protected Routes

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/chat-application.git
```

### Install Backend

```bash
cd backend
npm install
npm start
```

### Install Frontend

```bash
cd frontend
npm install
npm run dev
```
